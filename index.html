Interfejs użytkownika

Toolbar – Pasek narzędziowy

Cofnij (Undo) – cofa zmiany. Wiele czynności wykonanych w GitKraken można cofnąć. Jeśli akcja jest nieodwracalna, przycisk Cofnij będzie wygaszony
(Redo) – powtórz cofniętą zmianę. Jeżeli niepotrzebnie wycofamy jakąś zmianę, za pomocna przywrócić stan sprzed cofnięcia
Pobierz (Pull/Fetch) - Za pomocą tego przycisku można pobierać zmiany ze zdalnych repozytoriów. Jeżeli obok przycisku znajduje się ▼ można za jego pomocą uściślić sposób pobierania: do wyboru są następujące opcje:
  • Fetch All – pobierz wszystkie zmiany bez scalania, aktualizuje kopię zdalnego repozytorium
  • Pull (fast-forward if possible) – pobiera zmiany (fetch) i próbuje je scalić (merge) z lokalnym repozytorium
  • Pull (fast-forward only): odpowiednik polecenia: git fetch git merge --ff-only
  • Pull (rebase) – odpowiednik polecenia: git fetch git rebase
Wyślij (Push) – wysyła zmiany do zdalnego repozytorium zgodnie z ustawieniami wysyłania
Gałąź (Branch) – tworzy gałąź w lokalnym repozytorium
Odłóż na półkę (Stash) – Odkłada na półkę zmiany nad którymi pracujesz
Pobierz (zdejmij) z półki (Pop Stash) – Pobiera zmiany odłożone wcześniej na półkę (usuwając zmiany z półki) do lokalnego katalogu
Magazyn dużych plików (Large File Storage) – Jeżeli w repozytorium znajdują się pliki o dużej objętości i w repozytorium jest włączona obsługa dużych plików pojawi się przycisk LFS
Otwórz tablicę z zadaniami (Glo Issue Baoards) – otwiera dodatkową aplikację, w której można planować pracę, kolejne wydania wersji, scalanie zmian i tym podobne.
