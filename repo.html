Repo – pozwala na wyszukiwanie operacji związanych z repozytorium:

  • Tab – zamknięcie zakładki z repozytorium
  • Open Repo – otwarcie repozytorium
  • Open in file manager – otwarcie repozytorium w Menedżerze plików
  • Open in External Diff/Merge Tool – otwarcie porównywania zmian lub ich scalania w zewnętrznym progranie
  • Open in terminal – otwiera zewnętrzny domyślny terminal Git zdefiniowany w ustawieniach
  • Repo Management: Clone – klonuje zdalne repozytorium
  • Repo Management: Init – powoduje inicjalizację repozytorium w bieżącym katalogu
  • Repo Management: Open – otwiera lokalne repozytorium
